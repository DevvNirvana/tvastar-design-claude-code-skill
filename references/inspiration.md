## The Awwwards Design Formula

Sites that consistently win share these 7 traits:

1. **One exceptional typographic moment** — H1 that feels oversized and committed (56-96px desktop)
2. **One unexpected interactive element** — cursor effect, scroll reveal, 3D card, image trail
3. **Extreme whitespace confidence** — section padding that feels "too generous": 96-128px min
4. **Complete color discipline** — 2-3 colors max, accent applied to ONE thing only
5. **Consistent spatial rhythm** — same spacing units throughout (8px system, never random 13px)
6. **Motion that communicates** — every animation answers: where did it go? did my action work?
7. **Asset quality** — photography, illustration, or 3D is genuinely exceptional

## What to Study on Any Award-Winning Site (15-minute process)

1. **0.5 second test** — Cover the text. What's the dominant visual? That's their hierarchy.
2. **Type scale ratio** — Measure H1px ÷ body px. Winners: 3:1 to 6:1 ratio. Under 2:1 = too timid.
3. **Color count** — Open DevTools, check CSS variables. Award winners: 2-3 max. Where is accent?
4. **Section spacing** — What's the largest vertical gap? That's their confidence unit. Usually 80-128px.
5. **The delight moment** — What's the ONE thing that made you stop? What principle does it use?
6. **Mobile execution** — Check on phone. Does the hero text wrap gracefully at 375px?

## Color Trends 2026-2026

### Dominant now:
- **AI Indigo** — #6366F1 → #8B5CF6. Every AI startup uses this. Consider differentiating.
- **Dark Premium** — near-black (#09090B, NOT #000000) with violet/cyan/amber accents
- **Warm Minimal** — cream (#FFFBF5) + warm browns/amber. Reaction against cold tech blues.
- **Mature Glass** — restrained backdrop-blur (10-20% opacity), not frosted-glass on everything
- **Organic Aurora** — color blob mesh gradients, no hard edges, feels alive not designed

### Receding:
- Neon on pure black (except gaming/crypto — contextually correct there)
- Heavy glassmorphism on every card (2021 trend, now overused)
- Neumorphism (still impractical + low contrast)
- Flat gradient backgrounds (unless strongly brand-justified)

### Typography 2026-2026:
- **Oversized display** — 80-120px hero on desktop. The commitment to scale is the statement.
- **Variable fonts in motion** — weight morphs on scroll/hover (Fraunces wght axis)
- **Display serifs returning** — Fraunces, Melodrama, Playfair for editorial sections within tech sites
- **Geometric grotesque dominant** — Plus Jakarta, Outfit, Space Grotesk, Bricolage Grotesque
- **Intentional mono** — Space Mono, Geist Mono for code/technical context only

## The Senior Designer Eye Test

Run this before delivering any design:

| Test | Pass condition |
|------|---------------|
| **Scale test** | H1 and body text feel dramatically different in size |
| **Whitespace test** | Every section has room to breathe. Double the padding. |
| **Hierarchy test** | Cover text — visual weight alone shows what's most important |
| **Color discipline** | CTA color appears ONLY on the CTA. Nowhere else. |
| **Motion purpose** | Remove any animation — does UX suffer? If no, cut it. |
| **Mobile test** | Hero text wraps gracefully at 375px. Tap targets ≥ 44px. |
| **B&W test** | Print in grayscale — hierarchy still clear via layout alone? |
| **Von Restorff test** | Is there exactly ONE thing that breaks the visual pattern? |

## Practical Research Workflow (for any new design request)

1. **Identify category** (SaaS, e-commerce, portfolio, dashboard, etc.)
2. **lapa.ninja** → filter by category → study 3 examples, note: spacing, type weight, color count
3. **godly.website** → find motion reference in same category
4. **saaslandingpage.com or land-book.com** → 2-3 hero section references
5. **awwwards.com/websites** → filter by style keyword → find one aspirational reference
6. **Design with these internalized** — not copied, but absorbed
